# RAG-Question-Answer-System
•	Developed a Retrieval-Augmented Generation-based Question Answering system using LangChain.
•	Integrated IBM Watsonx Granite Large Language Models and RetrievalQA chain to generate context-aware responses
•	Implemented ChromaDB vector database with IBM Slate embedding model to create and retrieve semantic document embeddings.
•	Built interactive web-based user interface using Gradio for document upload and query execution.
